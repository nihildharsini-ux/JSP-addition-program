JSP Addition Program

Objective:

To perform addition inside a single JSP file by mixing HTML form + Java logic.

Description:
User enters two numbers in a form.
The same JSP file reads parameters using request.getParameter(), converts them to integers, and prints the sum.

•	Single JSP file
•	HTML form for number1, number2
•	Java scriptlet to calculate sum
•	Output displayed on the same page
